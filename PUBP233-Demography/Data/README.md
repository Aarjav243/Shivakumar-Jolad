# Data — PUBP 233: Introduction to Demography

Datasets organized by course topic. Duplicate copies (files that existed both in the root and in subfolders) and AutoRecovered files were removed on 2026-07-07 after checksum verification.

| Folder | Contents |
|---|---|
| 01-Population-Structure-Growth | Population pyramids, structure exercises, age standardization, decadal population (1971–2011), India/Japan population by sex 1970–2020 |
| 02-Fertility-Nuptiality | Student fertility data, Bangladesh cohort marriage life table |
| 03-Mortality | Age-specific death rates, WHO Mortality Database, UK student mortality data, IMR in Indian states (Data For India) |
| 04-Life-Tables | Country life tables (13 countries), WHO life table datasets, student exercise workbooks |
| 05-Migration-Census2011 | Census 2011 D-series migration tables — India, Delhi, Maharashtra (D01–D04) |
| 06-World-Population-Prospects | UN WPP 2020/2022/2024 population by 5-year age groups |
| 07-GBD-Global-Burden-of-Disease | IHME GBD 2023 mortality tables (S1–S19), disability weights, Lancet 2025 paper. IHME table filenames kept as-is for citation |
| 08-NFHS-STATA-Replication | STATA .do replication files and data (filenames kept as-is — scripts reference each other) |
| 09-Class-Exercises-Misc | Class survey, final exam workbook, Poston textbook extracted tables |
