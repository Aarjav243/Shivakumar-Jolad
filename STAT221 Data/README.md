# Data — STAT 221

Datasets and class workbooks, organized by course topic.

| Folder | Contents |
|---|---|
| 01-Data-Visualization | Anscombe quartet, Iris dataset, life expectancy vs GDP (Our World in Data), in-class visualization workbooks |
| 02-Central-Tendency | Central tendency distributions, student exercises, Krieg textbook tables |
| 03-Dispersion | Measures of dispersion workbooks, student exercises, Krieg Ch4 workbook |
| 04-Probability-Normal | Normal distribution intro (Khan Academy) |
| 05-Hypothesis-Testing | Z-test and t-test workbook |
| 06-Income-Inequality | Simulated income data (50 people; low/medium/high inequality) |
| 07-Surveys-Census-NFHS | Census 2011 questionnaires, NFHS-5 tables, digitized opinion-survey tables |
| 08-Class-Exercises | Pivot table exercises, trading data, calendar files (2022) |

Naming convention: `Topic-Description[-Source][-Date].ext`
